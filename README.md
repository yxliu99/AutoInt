# AutoInt
The code of IJCAI 2023 paper "Learning to Binarize Continuous Features for Neuro-Rule Networks".  
This is a `pytorch` implementation of AutoInt model.

### Prerequisites
- Python 3.7
- Pytorch>=1.4.0
